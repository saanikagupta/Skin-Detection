# Skin-Detection
Detect skin in images using computer vision.

### Code Requirements
The code is in Python (version 3.6 or higher will work).

### Dependencies
- import cv2
- import imutils
- import numpy

### Execution
To run the code, type ```python skindetector.py```
```
python skindetector.py
```
